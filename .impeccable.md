# .impeccable.md — Design Context

## Design Context

### Users
학술대회 현장 참석자. 발표 전후 혹은 세션 중 **모바일 기기**로 초록집(Abstract Book)을 열람한다. 읽기 속도와 탐색 편의성이 최우선이며, 논문·발표 자료를 빠르게 찾고 집중해서 읽는 것이 핵심 과업이다. PC에서도 사용하지만 주 경험은 모바일 세로 화면.

### Brand Personality
**세련 · 절제 · 학술적**

- 학회지나 고품질 저널처럼 권위 있지만 접근성 높은
- 과하지 않고 콘텐츠 자체가 돋보이는
- 인쇄물 느낌의 정갈함을 디지털로 옮긴 듯한 감각

### Aesthetic Direction
| 항목 | 방향 |
|------|------|
| 컬러 팔레트 | 딥 네이비(`#1a2744`) 주색, 오프화이트(`#f8f7f4`) 배경, 차콜(`#1c1c1e`) 텍스트 |
| 보조색 | 옅은 네이비 틴트(`#e8edf5`), 골드 포인트(`#c9a84c`) for 강조 |
| 다크모드 | 지원 (시스템 설정 + 수동 토글 모두) — 어두운 배경은 `#12172b` |
| 폰트 | NanumSquare (이미 적용) — 제목에 무게감 있는 800 weight |
| 여백 | 넉넉한 padding, 행간 1.7~1.8 — 인쇄물 레이아웃 감각 |
| 반경 | 카드는 `rounded-2xl`(16px), 버튼은 `rounded-xl`(12px) — 너무 둥글지 않게 |
| 그림자 | 절제된 `shadow-sm` 또는 `shadow-md` — 과도한 그림자 금지 |
| 이미지/아이콘 | Heroicons 유지, 선 굵기 균일하게 |

**Anti-references (피해야 할 것)**
- 알록달록한 SaaS 대시보드 느낌 (현재 파란색/보라색 그라디언트)
- 과도한 카드 애니메이션, 호버 이펙트 남발
- 소비자 앱 같은 컬러풀한 배너

### Design Principles
1. **콘텐츠 우선** — UI 크롬은 최소화, 초록 텍스트가 화면의 주인공이어야 한다
2. **모바일 퍼스트** — 터치 타겟 44px 이상, 스크롤 흐름 중심의 레이아웃
3. **학술적 권위** — 딥 네이비 팔레트와 절제된 타이포그래피로 신뢰감 표현
4. **최소 마찰** — 콘텐츠까지 최소 탭 수, 탐색 구조 단순화
5. **인쇄물 감성** — 충분한 여백, 규칙적인 그리드, 산만한 장식 요소 배제

### Tech Context
- React 18 + TypeScript + Vite
- Tailwind CSS v3 (darkMode: "class")
- Firebase (Firestore + Hosting)
- PWA (vite-plugin-pwa)

**파일:** `src/App.tsx`


- `/viewer/:publicationId` - 뷰어 (읽기)

## 📝 기술적 결정 사항

### 1. 완전 제거 vs 기능 유지
**결정:** 오프라인 편집 기능 완전 제거
**이유:**
- 실제 사용 시나리오에서 필요성 낮음
- 구현 복잡도에 비해 효과 적음
- 데이터 동기화 오류 위험 회피

### 2. OnlineOnly 컴포넌트 위치
**결정:** `src/components/auth/`에 배치
**이유:**
- ProtectedRoute와 유사한 목적
